# NUM.lsp — תוסף מספור אובייקטים ל-ZWCAD

## תיאור כללי
תוסף
AutoLISP
למספור אוטומטי של אובייקטים בשרטוט הנדסי.
מספר פוליגונים סגורים (
LWPOLYLINE
) ועיגולים (
CIRCLE
), מסדר אותם מימין לשמאל ומלמעלה למטה, ויוצר טקסט ממוספר ליד כל אחד.

---

## פקודות

| פקודה | תיאור |
|---|---|
| `NUM` | פקודה ראשית — בפעם הראשונה פותחת הגדרות, אחר כך תפריט |
| `NUMSET` | קיצור לפתיחת חלון ההגדרות |
| `NUMCHECK` | קיצור לסריקת מספרים |

### תפריט NUM
```
[Settings / Objects / Window / Check]
```
- **Settings** — פתיחת חלון הגדרות
- **Objects** — בחירה ידנית של אובייקטים
- **Window** — בחירה במרובע (משמאל לימין = חלון, מימין לשמאל = חוצה)
- **Check** — סריקת שלמות המספרים

## הגדרות (נשמרות פר קובץ DWG)

| שדה | תיאור | ברירת מחדל |
|---|---|---|
| שכבת טקסט | השכבה שבה יוצרים טקסטים | `NUM_TEXT` |
| גובה טקסט | גודל הטקסט ביחידות | `2.5` |
| סגנון טקסט | פונט / סגנון | `Standard` |

| סיומת | טקסט אחרי המספר | ריק |
| מספר התחלה | המספר הראשון בסדרה | `1` |

- **CIRCLE** (עיגול)

קווים, תשרים, בלוקים — מתעלמים.

## התנהגות המספור

### מיון
מימין לשמאל, מלמעלה למטה.
סבילות שורה: `max(5.0, גובה-טקסט × 2)`.

### אובייקט שכבר ממוספר
- אם הטקסט **קיים על המסך** → שואל "למספר מחדש? [Yes/No]"
- אם הטקסט **נמחק** → ממשיך ישירות ללא שאלה
- הטקסט הישן נמחק מהמסך לפני יצירת החדש

### עותקים
עותק של אובייקט ממוספר מקבל את אותו המספר.
כאשר ההגדרות משתנות — כל הטקסטים (כולל עותקים) מתעדכנים אוטומטית.

## XData — מבנה התיוג

### על ישות הטקסט (`NUM_TAG`):
```
(1000 . version)       ; גרסה
(1000 . num-str)       ; המספר כמחרוזת
(1000 . orig-handle)   ; handle של האובייקט המקורי
```

### על האובייקט המקורי (`NUM_TAG`):
```
(1000 . version)       ; גרסה
(1000 . num-str)       ; המספר כמחרוזת
(1000 . txt-handle)    ; handle של הטקסט
```

---

## פונקציית Check

1. **סורקת** כל אובייקט ממוספר — בודקת אם הטקסט שלו עדיין קיים
2. **מציגה** סיכום: כמה אובייקטים, כמה טקסטים קיימים, כמה חסרים
3. **מציעה שחזור** למספרים שנמחקו
4. **בודקת פערים** ברצף מהמספר המוגדר ועד המקסימום
5. **מציעה מילוי** של פערים על ידי בחירת אובייקטים
6. **מציעה איפוס מלא** — מחיקת כל המספרים ותחילה מחדש
